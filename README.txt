
C言語を用いて電卓を開発したものです。

C言語でデスクトップアプリを開発するためにDXライブラリを使用しています。

DXライブラリのURL
https://dxlib.xsrv.jp/

DXライブラリの環境の整え方URL
https://dxlib.xsrv.jp/dxuse.html

上記の設定をしなくてもexeファイルは動作します。


exeファイルを起動することで電卓が起動します。
cppファイルにはexeファイルのソースコードが記載されています。

-----------------------------
電卓の使い方
マウスで画面をクリックする、またはキーボードから数値や演算子を入力したり計算を実行することができます。

終了する際は画面の上部（計算式が表示される場所）をクリックするか、キーボードのESCAPEキーを入力することで画面を閉じることができます。

キーボード入力の際のリファレンス
0〜9	:各数字のキー
. - /	:各記号のキー
+ *	:（SHIFTキーを押さずに）各記号のキー
(	:Lキー
)	:Rキー
%	:Pキー
=	:ENTERキー
終了	:ESCAPEキー
