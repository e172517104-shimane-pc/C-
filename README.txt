
C言語を用いて電卓を開発したものです。

C言語でデスクトップアプリを開発するためにDXライブラリを使用しています。

DXライブラリのURL
https://dxlib.xsrv.jp/

DXライブラリの環境の整え方URL
https://dxlib.xsrv.jp/dxuse.html

上記のホームページの説明に従い環境を整えて、コードをコピペすることで実行が可能になります。

-----------------------------
電卓の使い方
マウスで画面をクリックする、またはキーボードから数値や演算子を入力したり計算を実行することができます。

終了する際は画面の終了ボタンをクリックするか、キーボードのESCAPEキーを入力することで画面を閉じることができます。
